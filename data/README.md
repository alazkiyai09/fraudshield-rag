# Data Notes

`data/sample/` contains synthetic records for local testing and demos.

- `sample_transactions.csv`: synthetic transaction events with fields used for fraud-pattern queries.
- `sample_case_notes.txt`: synthetic analyst notes for free-text ingestion.
- `sample_compliance_report.pdf`: optional placeholder path from spec; replace with real reports for realistic retrieval quality.

Do not upload production PII or protected banking records to local demo environments.
